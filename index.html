ℹ️ 关于这个工具（点 × 收起，以后从底部展开）
  • 📥 持仓数据存在 你自己浏览器的 localStorage，从不上传服务器
  • 👥 其他人打开同一个网址，看到的是 他们自己 的数据，互相不可见
  • 💾 清缓存 / 换浏览器 / 换设备会丢失 → 点 💾 导出备份 存 JSON 文件
  • 📂 在新浏览器恢复 → 点 📂 导入备份 选那个 JSON
  • 🔗 跨设备自动同步（推荐 Chrome/Edge）：把 JSON 放 Google Drive 文件夹，点 🔗 文件同步 选它 → 改动自动写文件 → Drive 自动同步到云 → 其他设备打开同一个文件即可
  • 👤 改用户名（左上角）方便区分多浏览器/多设备的数据
