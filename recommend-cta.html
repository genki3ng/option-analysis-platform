主 CTA · highlight 候选

「包租公推荐指数」是产品王牌。当前金色填充已经做了基础视觉权重，但跟「复盘报告」的对比还不够"王牌"。 下面 4 个 highlight 方向，全部带 hover 状态，可以鼠标移上去看。 注意有动画的方案 (A / B) 在静止图里看不出来。

CURRENT
当前版

金色 linear-gradient 填充 + 暖金阴影晕 + hover brightness 微亮。中规中矩。

OPTION A
呼吸光环 · Breathing Halo

基础保持当前金色填充，外面加一圈金色 box-shadow 做 3.2 秒的呼吸动画（pulse 出去到 9px 然后淡入）。视觉上"自带光环"。hover 时呼吸暂停。

OPTION B
高光扫过 · Light Sweep

基础保持金色填充，加一条对角的白色高光带，每 4.8 秒从左滑到右。模仿 iOS app icon 在 spotlight 下的反光感。动效轻、不喧宾夺主。

OPTION C
金属质感 + 王牌徽章 · Metallic + Ace Badge

多层 gradient 营造金属反光（暗→亮→暗），1.5px 深金边框，右上角悬一个深色 ★ ACE 徽章。静态、不动，但视觉权重明显更高，"质感"先到。

OPTION D
双层光晕 · Glowing Halo Ring

按钮外圈"隔空"再加一层金色光环（4px 间隙 + 1px 金色 ring + 大范围模糊光晕）。视觉上像被光照、悬浮在页面上。静态不动，但"光感"非常强。

4 个方向各有侧重：A/B 带轻量动画 / C 加徽章信号 / D 完全靠静态光晕。 选定后告诉我，我落地到 /app 的 .rec-btn-large 上。
